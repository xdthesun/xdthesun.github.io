git合并分支并删除分支多余分支

发表于 2019-02-26 | 分类于 git

有时候我们需要在本地拉一个dev分支用来开发，开发完成后并不是直接提交到服务器，而是在本地合并到master分之后提交到服务器，这样就不会在服务器显示多余的分支，而大多数时候是在提交到服务器再merge到对应分支，这样在代码仓库里是能看到对应的分支。
